# Blog Post Writing Rules

1. **Short paragraphs**: Keep every paragraph to 2-3 sentences maximum. Readers skim online content, so dense paragraphs will lose them. No exceptions.

2. **Thorough explanations**: Each point you make must be thoroughly explained with supporting evidence, examples, and data. A point without evidence is just an opinion. Include at least one statistic or real-world example per point.

3. **Word count**: Keep the total post to 400 words. Brevity is key for online readers.

4. **Five points minimum**: Cover at least 5 distinct points or reasons in the post.

5. **Conversational tone**: Write as if speaking to a friend. Use "you" and "your" frequently.

6. **No filler**: Every sentence must add value. Remove any sentence that doesn't directly support a point.

7. **Specific examples**: For each point, include a specific named example (company, tool, person, or event). Generic examples don't count.

8. **Opening hook**: Start with a surprising statistic or bold claim.

9. **Ending**: End with a question to the reader.

10. **No jargon**: Write for a general audience. If you must use a technical term, define it inline.
