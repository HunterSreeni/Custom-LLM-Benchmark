# Blog Post Writing Rules - ALL Must Be Followed

1. **Word count**: 500-700 words. Not one word under 500 or over 700.
2. **No em dashes**: Never use - (em dash). Use a regular hyphen (-) instead. Every single time.
3. **Prose-only body**: No bullet points, no numbered lists in the body text. The only exception is if you include a "Key Takeaways" box at the very end.
4. **Conversational tone**: Write as if talking to a friend over coffee. Use "you" and "your" at least 8 times.
5. **Title format**: Title must be under 60 characters and include the primary keyword.
6. **Opening hook**: First sentence must be a question or a bold statement (not "In this post...").
7. **Paragraph length**: Every paragraph must be 2-4 sentences. No single-sentence paragraphs except for emphasis (max 1 per post).
8. **SEO keywords**: Include the primary keyword in the title, first paragraph, one subheading, and last paragraph. Include 2-3 secondary keywords naturally.
9. **Affiliate link**: Include exactly one Amazon Associates link with tag `huntersreen05-21`. It must feel natural, not forced. Recommend a specific product by name.
10. **Ending**: End with a question directed at the reader. The question must relate to the main topic and invite a comment.
11. **Subheadings**: Use exactly 3 H2 subheadings to break up the post. Each subheading should be under 50 characters.
12. **Image placeholders**: Include exactly 2 image placeholders in the format `[IMAGE: description of suggested image]`.
13. **Internal link placeholder**: Include one `[INTERNAL LINK: related post topic]` placeholder.
14. **Meta description**: Write a 150-160 character meta description at the top of the post.
15. **No cliches**: Do not use these phrases: "game-changer", "at the end of the day", "it goes without saying", "in today's fast-paced world", "without further ado".
